# Luxe Ledger — Airbnb Finanzas

Control de finanzas premium para tu propiedad en Airbnb.  
Esta versión utiliza **LocalStorage** para la persistencia de datos, eliminando la dependencia de servicios externos como Firebase.

## Características
- Gestión de ingresos y gastos por mes.
- Gráficas de rendimiento dinámicas.
- Predicción de ingresos basada en historial.
- Modo oscuro y diseño responsive.
- Exportación de informes en CSV.
